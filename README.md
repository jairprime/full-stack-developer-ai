# 🧑‍💻 Full-Stack Developer AI

## 📋 Overview

**Full-Stack Developer AI** is a personal learning and practice repository for exploring full-stack development concepts, algorithms, and problem-solving. It hosts small projects and exercises that cover both low-level programming and modern web development, serving as a portfolio of growth across the software engineering spectrum.

From a terminal-based calculator in C to future web applications, this repo documents the journey of building foundational skills in logic, data structures, APIs, and user interfaces.

## ✨ Projects

- **Calculator** (`calculator/`): A command-line calculator written in C that supports addition, subtraction, multiplication, and division with basic error handling for division by zero.
- *More projects coming soon — covering frontend, backend, and full-stack integrations.*

## 🛠️ Technologies

- **Languages**: C, JavaScript (planned), TypeScript (planned)
- **Frontend** (planned): React, HTML, CSS
- **Backend** (planned): Node.js, Express
- **Database** (planned): PostgreSQL
- **Tools**: Git, GCC, VS Code

## 📦 Installation

### Prerequisites

- GCC (GNU Compiler Collection) or any C compiler
- Git

### Clone the repository

```bash
git clone https://github.com/jairprime/full-stack-developer-ai.git
cd full-stack-developer-ai
```

### Build and run the Calculator

```bash
cd calculator

# Compile
gcc main.c -o program

# Run
./program
```

## 🚀 Usage

1. Navigate to any project folder inside the repository.
2. Follow the specific instructions in that project's own documentation (if available).
3. For the **Calculator** project:
   - Run the compiled binary.
   - Enter an operator (`+`, `-`, `*`, `/`) when prompted.
   - Enter two numbers.
   - View the result.
